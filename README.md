# g-url
Doi Shortlink URL (Backend Only)

## Cara menjalankan server : 
- npm install
- npm run dev

## Cara menjalankan dokumentasi :
- cd documentation/
- npx serve


## Dibuat menggunakan :
- NodeJS (ExpressJS)
- MongoDB (Mongoose)
- Insomnia-Documenter (API Documentation)
- Joi (validator)
- Nodemailer (Google OAuth2 SMTP)
- JSON Web Token (Auth)
- Docker
